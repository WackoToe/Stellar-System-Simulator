#Stellar System Simulator (S3)
### Brief description
Stellar system simulator(S3) is a simulator based on physics realised as a final term project for the Human-Computer interaction exam.
S3 is based on html, css and javascript.

### Functionality
S3 allows you to:
* insert planet
* delete planet
